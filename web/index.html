Get Started in 4 Steps

1

Install gl2gh

pip install gl2gh or clone the repository and run pip install -e ".[dev]"

2

Inspect Your Pipeline

Run gl2gh inspect .gitlab-ci.yml -v to see your pipeline structure, jobs, stages, and templates.

3

Migrate

Run gl2gh migrate .gitlab-ci.yml -o .github/workflows to generate GitHub Actions workflows. Add --ai for complex pipelines.

4

Validate and Push

Run gl2gh validate .github/workflows/ then commit and push. Use gh workflow run to test.

Ready to Migrate?

gl2gh is open source and free to use. The rule-based converter works instantly with no API keys. Enable AI mode for complex pipelines.
