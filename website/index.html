Migrate from GitLab CI to GitHub Actions

A production-grade CLI tool that converts your GitLab CI/CD pipelines to GitHub Actions workflows. Rule-based conversion with optional AI enhancement powered by GitHub Copilot.

Everything you need to migrate

From simple pipelines to complex multi-stage deployments, gl2gh handles the conversion so you can focus on shipping.

⚙

Rule-Based Converter

Deterministic mapping of 30+ GitLab CI features to GitHub Actions. Handles stages, variables, cache, artifacts, services, environments, matrix builds, and more with zero configuration.

✨

AI-Enhanced Migration

Optional GitHub Copilot-powered agent for complex patterns. Handles advanced rules conditions, multi-project pipelines, template resolution, and edge cases that pure rules cannot cover.

Powerful CLI

Five commands for the full migration workflow: migrate, inspect, validate, migrate-repo, and gh-status. Rich terminal output with syntax highlighting, tables, and progress indicators.

⇄

Repository Migration

End-to-end repository migration from GitLab to GitHub. Integrates with the GitHub CLI for repo creation, workflow management, and deployment verification.

How it works

Three steps to migrate your CI/CD pipeline.

1

Inspect

Analyze your GitLab CI pipeline to understand its structure: stages, jobs, templates, services, and variables. Identify complex patterns before converting.

2

Migrate

Run the converter to generate GitHub Actions workflow files. Use rule-based mode for speed, or enable AI enhancement for complex pipelines.

3

Validate

Check the generated workflows for correctness, security issues, and best practices. Push to GitHub and verify your workflows run successfully.

Quick start

Install and run your first migration in under a minute.

Terminal 
# Install gl2gh
pip install -e ".[dev]"

# Inspect your pipeline
gl2gh inspect .gitlab-ci.yml -v

# Preview the conversion (dry run)
gl2gh migrate .gitlab-ci.yml --dry-run

# Run the migration
gl2gh migrate .gitlab-ci.yml -o .github/workflows

# Validate the output
gl2gh validate .github/workflows/

# (Optional) AI-enhanced migration
export GITHUB_TOKEN="ghp_your-token-here"
gl2gh migrate .gitlab-ci.yml --ai
